# Templates and Voice Modes

Choose one template and one voice mode.

## Templates

### limited-drop

Use for opening, coupon, flash sale, or quick-start campaigns.

Flow: fast hook -> product reason -> provided benefit -> link/action.

### holiday-stockup

Use for Spring Festival, Mid-Autumn, Dragon Boat, family dinner, return home, gift, or gathering.

Flow: occasion reminder -> why this product fits -> taste/scene fact -> action.

### return-sale

Use for restock, return, missed last time, group asked for it.

Flow: missed-it hook -> why it returned -> confirmed facts -> action.

### final-call

Use for deadline, last day, final reminder, soon ending.

Flow: time pressure -> strongest confirmed reason -> direct action.

### trace-tail

Use for origin, source, production, seasonal tail goods.

Flow: source/process fact -> why this batch matters -> tail/ending fact -> action.

### restock-tail

Use for added stock, tail stock, sold out when gone.

Flow: restock notice -> product position -> confirmed stock/deadline fact -> link/action.

## Voice Modes

- `official-warm`: clear, warm, restrained; best default for official account.
- `service-notice`: short, factual, service-like; best for links and rules.
- `old-group-owner`: familiar, direct, slightly urgent; best for WeChat groups.
- `story-selling`: process and context; best when user provides origin or negotiation facts.

## Length Control

- Sparse facts: 8-14 lines.
- Normal facts: 12-28 lines.
- Many links or price tiers: keep copy short and list links cleanly.
