# Quality Gate

Before final output, check:

- If confidence is A, the named wine is supported by visible text, distinctive evidence, or user confirmation.
- If confidence is B, the chat uses uncertainty language and does not write the wine name as fact.
- If confidence is C, the chat does not mention any specific wine brand or series.
- No invented price, discount, batch, stock, gift, or sales claim.
- At least 4 roles and 6-10 total lines.
- Lines are short and uneven; no tasting-note paragraph.
- No title, explanation, scene card, or image placeholder in ordinary output.

Rewrite once if any hard gate fails.
